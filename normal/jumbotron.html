Bootstrap 5 jumbotron

In Bootstrap 3, a jumbotron was introduced as a large, padded box for highlighting special content or information.
Now, jumbotrons does not supported by Bootstrap 5 version.
We can achieve the same result with an element, special helper classes, and a color class.
